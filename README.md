# Recipin: Your Digital Cookbook

Recipin is a modern and intuitive web application designed for food lovers to discover, create, and share their favorite recipes. Built with a clean and responsive user interface, it provides a seamless experience for browsing, saving, and managing recipes.

## Key Features

-   **User Authentication:** Secure user registration and login system.
-   **Create & Share:** Easily create detailed recipes with ingredients, instructions, and photos.
-   **Recipe Feed:** Discover new and exciting recipes from the community.
-   **Save Favorites:** Bookmark your favorite recipes for quick access later.
-   **Detailed View:** See all the details of a recipe on one page, including cooking time, difficulty, and ingredients.
-   **Search & Filter:** Quickly find the recipes you're looking for.

## Tech Stack

-   **Frontend:** React, Vite, React Router
-   **Backend:** Node.js, Express
-   **Database:** PostgreSQL
-   **Authentication:** JWT (JSON Web Tokens) & bcrypt
-   **Styling:** CSS with Lucide React for icons

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

-   [Node.js](https://nodejs.org/)
-   [npm](https://www.npmjs.com/get-npm)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/recipin.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```

---
